### SmartLock Web Client

## Prerequesting 
- node.js
- npm
- yarn
- git


## Installation

```bash
$ git clone https://github.com/akasaa101/SmartLockWebClient.git

```

```bash
$ yarn install
```

```bash
$ yarn start
```
